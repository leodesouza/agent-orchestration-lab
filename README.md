# Agent Orchestration Lab

Hands-on experiments with modern AI agent orchestration frameworks.

This repository is part of the AI Systems Lab project and explores how enterprise AI agents are designed, coordinated and deployed.

The goal is to understand the engineering principles behind production-ready AI systems rather than focusing only on prompting.

## Technologies
- Python
- LangGraph
- CrewAI 
- AutoGen 
- OpenAI
- Anthropic 
- Ollama
- Docker
